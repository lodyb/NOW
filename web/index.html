Bot Commands

These commands can be used in Discord:

  • NOW play [search term] - Play a media file
  • NOW play [search term] {amplify=2,reverse=1} - Apply effects
  • NOW clip=4s [search term] - Create a clip
  • NOW quiz - Start a music quiz
  • NOW quiz {reverse=1} - Quiz with effects
  • NOW stop - Stop a running quiz
